Configuration Examples

Default Height (75vh)

initializeIzmo({ enabledFeatures: [...], // No widgetHeight specified - uses default 75vh });

Small Height (50vh)

initializeIzmo({ enabledFeatures: [...], widgetHeight: "50vh" });

Large Height (90vh)

initializeIzmo({ enabledFeatures: [...], widgetHeight: "90vh" });

Custom Pixel Height (400px)

initializeIzmo({ enabledFeatures: [...], widgetHeight: "400px" });

Supported Height Values

  • Viewport units: 50vh, 75vh, 90vh, etc.
  • Pixel values: 300px, 400px, 500px, etc.
  • Percentage: 50%, 75%, 100%, etc.
  • CSS calc(): calc(100vh - 100px), etc.

Default: If no widgetHeight is specified in the config, the widget defaults to 75vh (75% of viewport height).
