Izmo Accessibility Widget - WCAG 2.1 AA Compliant Demo

This page automatically loads the enhanced Izmo Accessibility Widget with comprehensive WCAG 2.1 Level AA compliance features. The widget provides various accessibility features to improve the web experience for all users.

WCAG 2.1 AA Compliant Features

The accessibility widget now includes comprehensive WCAG 2.1 Level AA compliant features:

  • Skip Links: Navigate quickly to main content sections
  • Keyboard Navigation: Enhanced keyboard-only navigation support
  • Focus Enhancement: Improved focus indicators for better visibility
  • Color Blind Support: Filters and patterns for color vision deficiencies
  • Reduced Motion: Disable animations for motion sensitivity
  • Screen Reader: Reads out text content when you click on elements
  • Contrast: Adjusts background and text contrast for better visibility
  • Highlight Links: Makes links more visible and identifiable
  • Saturation: Controls color saturation for visual comfort
  • Bigger Text: Increases text size for easier reading
  • Text Spacing: Adjusts letter and word spacing
  • Text Align: Changes text alignment for better reading flow
  • Line Height: Customizes line spacing for improved readability
  • Dyslexia Friendly: Applies dyslexia-friendly fonts and spacing
  • Dictionary: Provides word definitions on demand
  • Page Structure: Visualizes and navigates page structure
  • Pause Animations: Stops or reduces motion for users with vestibular disorders
  • Hide Images: Conceals images to focus on text content
  • Cursor: Enhances cursor visibility and size
  • Tooltip: Provides enhanced tooltips and context information
  • Screen Mask: Creates screen masking for focused reading
